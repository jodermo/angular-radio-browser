## Installation

##### Angular (main app):
Run `npm install -g @angular/cli` to install Angular CLI.<br>
Run `npm install` to install dependencies.<br><br>
(To get more help on the Angular CLI use `ng help` or go check out the [Angular CLI README](https://github.com/angular/angular-cli/blob/master/README.md).)


##### Cordova (mobile app):
Run `npm install -g cordova` to install Cordova.<br>
Run `cd cordova-app && npm install` to install dependencies.

<br><br>

## Development server

Run `ng serve` for a dev server. <br>
Navigate to `http://localhost:4200/`.<br>
The app will automatically reload if you change any of the source files.

<br><br>

## Build

Run `ng build` to build the project.<br>
The build artifacts will be stored in the `dist/` directory.<br>
Use the `--prod` flag for a production build.

<br><br>

## Mobile App (Cordova)

Run `build:cordova` to build android APK.<br>
Run `build:cordova:release` to build android APK as release.<br>
Run `build:cordova:device` to build android APK and run at connected device.<br>
Run `build:cordova:device:log` to build android APK and run at connected device with logging output.
